# Pet Finder: Report and Find missing pets nearby.

Came across a wondeing pet? You can use this app to help your fellow neighbours find their wondering pets.

## The Problem
All of the pets like to escape and wonder around and this gets pet owner worried. So I designed this application which would allow users to report missing pets when they come across a wondering pet. So no more worries about your missing pet.

## Features
- No user registeration required
- Upload a photo
- Automatically captures the last seen location
- Notify nearby users
